Global Sequence Alignment

Dynamic Programming

Takes two strings representing unique genetic sequences. THe program calculates the "edit-distance" to recover the optimal alignment. With dynamic programming, the problem is broken into subproblems in which the alignment is found on suffixes of the original input.


GuitarHero

Symbol Tables, Karplus-Strong algorithm

Turns keyboard into the strings of a guitar using the Karplus-Strong algorithm and a symbol table to map keys. Each "GuitarString" object used a queue of frequencies to simulate the sinusoidal vibrations of a guitar string.


Pseudorandom Text Generator

Markov Models, Stochastic Processes, Symbol Tables

Reads input text, a integer k and output size T. The program takes a kgram-string of size k and simulates a stochastic process in which the frequency of each subsequent character is stored in a symbol table. After reading the input text, the program generates psuedo-random text using the probabilistic distribution of characters gleaned from the input text.

Sample input
Sample 1000 character long output using k-gram of size 7


Traveling Salesman Problem

Combinatorial Optimization, Greedy Heuristics, Linked Lists

Reads .txt file of points representing cities and generates a non-optimal tour that attempts to minimize the total distance traveled. Implemented with a linked list
